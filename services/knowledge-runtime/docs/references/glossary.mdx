## C

### Cross-language search

Cross-language search (also known as cross-lingual retrieval) is a feature introduced in version 0.19.0. It enables users to submit queries in one language (for example, English) and retrieve relevant documents written in other languages such as Chinese or Spanish. This feature uses the configured default chat model to translate queries and improve cross-language semantic matching.

By enabling cross-language search, users can access a broader range of information regardless of language barriers.

This feature is available in retrieval tests and dataset retrieval settings. See [Run retrieval test](../guides/dataset/run_retrieval_test.md) for further details.
